批量PDF翻译工具

简介

程序功能

把上百份包含图片的PDF翻译成中文文档，并保存为txt文件。

实现步骤

  1. 用户导入需翻译的PDF所在文件夹路径
  2. 把PDF每一页保存为图片
  3. 对每一张图片使用OCR工具提取文字，分别保存为txt
  4. 把所有txt汇总
  5. 用户输入AI的API密钥、地址和模型名
  6. 切割txt，每一份3万字
  7. 使用AI翻译每一份txt
  8. 汇总txt，保存为新的txt文件

环境搭建

  1. 安装库
  2. 安装OCR工具
  • Windows：从Tesseract官网下载安装
  • Mac：brew install tesseract
  • Linux：sudo apt install tesseract-ocr
  • 下载完自己配置环境变量
